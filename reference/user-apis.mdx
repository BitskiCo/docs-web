title: "Overview"
---

Bitski Account APIs can:

- get public information for a [Bitski profile](/reference/profiles)
- get on-chain information for a variety of use cases:
  - [ERC20 tokens and NFTs](/reference/get-balances) owned by a wallet
  - Get the [owner](/reference/get-balances) of a specific token
  - [Transaction history](/reference/get-activity) with enriched details
  - Metadata, for both [contracts](/reference/get-contract-metadata) and
    [tokens](/reference/get-token-metadata)

---

<Note>
**No auth needed**

You don't need an authentication token to use these endpoints.

</Note>
