### Wallet as a Service (WaaS)

Bitski provides APIs to easily and securely build full featured wallet
experiences that feel native to your platform.

<Card title="Wallet Creation APIs" icon="layer-plus" color="#4FA936" href="/wallet-as-a-service/wallet-creation-apis">
  Create a fully functioning Ethereum wallet for your users using your existing authentication system.
</Card>
<Card title="Wallet Experience APIs" icon="comment" color="#4FA936" href="/wallet-as-a-service/wallet-experience-apis">
  These APIs allow users to create rich wallet experiences to support one of the most NFT friendly wallet solutions.
</Card>
<Card title="NFT APIs" icon="hexagon-image" color="#4FA936" href="/wallet-as-a-service/nft-apis">
  Enable users to easily buy, mint and distribute NFTs with little to no coding experience.
</Card>
<Card title="Web3 APIs" icon="chart-network" color="#4FA936" href="/wallet-as-a-service/web-3">
  Sign transactions and interact with Web3 on any network.
</Card>
<Card title="SDK" icon="rocket" color="#4FA936" href="/wallet-as-a-service/sdk/sdk-overview">
  Quickly integrate wallets and web3 features into your product on web or mobile platforms.
</Card>
