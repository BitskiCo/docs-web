---
title: "POST Batch Floor Prices"
api: "POST https://api.bitski.com/v2/floor-prices/batch?supportedChainIds={chainIds}"
description: "Get batch floor prices for a wallet on a chain"
---

### Parameters

<ParamField query="supportedChainIds" type="array of string" placeholder="[1]" required>
  The ethereum chain ids to include
</ParamField>

### Examples

![Floor Price](/assets/images/batch-floor-prices.jpeg)

<RequestExample>

```bash Request
curl -X 'POST' \
  'https://api.bitski.com/v2/floor-prices/batch?supportedChains=1' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "contractIds": [
    {
      "address": "0x1C505d3AC1B902b69bc153aEdad477c9bC9ccde7",
      "chainId": 1,
      "coinType": 60
    }
  ]
}'
```

</RequestExample>
