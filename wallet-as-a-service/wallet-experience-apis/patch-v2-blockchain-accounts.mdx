---
title: "PATCH Blockchain Accounts"
api: "PATCH https://api.bitski.com/v2/blockchain/accounts/{accountId}"
authMethod: "bearer"
description: "Update a users blockchain account details"
---

### Parameters

<ParamField query="accountId" type="string" required />

<RequestExample>
```bash Request
curl -X 'PATCH' 'https://api.bitski.com/v2/blockchain/accounts/{accountId}' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <ACCESS_TOKEN>' \
  -d '{
  "displayName": "My Demo Wallet",
}'
```
</RequestExample>
