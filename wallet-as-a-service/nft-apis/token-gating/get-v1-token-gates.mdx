---
title: "GET Token Gates"
api: "GET https://api.bitski.com/v1/apps/{appId}/gates"
authMethod: "bearer"
description: "Get token gates"
---

<Note>Access to this API is available upon request to sales@bitski.com.</Note>

Limit access to web pages or other experiences to users who control a specific token.

![Example](/assets/images/gate-37fcd3c18c3afec1456c1d81dc6288cc.png "Gated Order")

<ParamField path="appId" type="string" required>
  Your Bitski Application ID
</ParamField>
