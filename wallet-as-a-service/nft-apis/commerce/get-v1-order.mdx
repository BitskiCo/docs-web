---
title: "GET Order"
api: "GET https://api.bitski.com/v1/orders/{orderId}"
authMethod: "bearer"
description: "Get details about a given order"
---

<Note>Access to this API is available upon request to sales@bitski.com.</Note>

### Path

<ParamField path="orderId" type="string" required>
  The order id
</ParamField>
