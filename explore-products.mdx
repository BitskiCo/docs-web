### Wallet as a Service (WaaS)


<Card title="Activity API" icon="comment" color="#4FA936" href="/wallet-as-a-service/activity-api/activity-api">
  See full parsed history for all actions taken on-chain. Includes ENS resolution and multi-effect activity details for complex activity types
</Card>

<Card title="Balances API" icon="money-bill" color="#4FA936" href="/wallet-as-a-service/balances-api/get-balances">
  Get the NFT and currency balances for a set of addresses across chains
</Card>

<Card title="Notifications API" icon="circle-info" color="#4FA936" href="/wallet-as-a-service/notifications-api/notifications-api">
  Get the data necessary to send push notifications or otherwise react when on-chain events happen to your users
</Card>

<Card title="Portfolio API" icon="briefcase" color="#EB9C3F" href="/wallet-as-a-service/portfolio-api/portfolio-api">
  Get statistics for a set of addresses across chains and over time. Includes currency balances, and NFT pricing through floor price information
</Card>

<Card title="Signing API" icon="pen" color="#4FA936" href="/wallet-as-a-service/signing-api/signing-api">
  Optionally prompt users for approval when signing. Transaction approval screen simulates transaction effects, parses complex transaction results, and displays content in human-readable terms
</Card>

<Card title="Token Details API" icon="hexagon-image" color="#EB9C3F" href="/wallet-as-a-service/token-details-api/get-contract-metadata">
  Get metadata about a given token, including floor prices, spam scores, and cached image and video assets
</Card>

<Card title="User API" icon="user" color="#4FA936" href="/wallet-as-a-service/user-api/get-a-public-profile">
  Retrieve info about Bitski users
</Card>
